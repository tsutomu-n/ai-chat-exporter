# ai-chat-exporter overview
- Purpose: Browser bookmarklet to export AI chat conversations from sites like ChatGPT, Claude, Grok, and Google AI Studio into Markdown or plain text.
- Stack: Readable source in `src/ai-chat-export.js`, shell generation scripts, Bun-based tests using `bun:test`.
- Public artifacts: Six language/browser-specific bookmarklet files in repo root plus mirrored copies under `docs/`, localized README files, localized docs guides, curated screenshots under `assets/`.
- Structure: `src/` readable source, `scripts/` generation/sync scripts, `tests/` repository and behavior tests, `docs/` public guide/assets mirror.
