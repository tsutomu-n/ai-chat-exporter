# アーカイブ

このフォルダには、現在の第一推奨ではない配布物をまとめています。

## いま使うファイル

- `../ai-chat-export.chrome.bookmarklet.oneliner.js`
  - Chrome / Chromium 系で使う現在の正式版
- `../ai-chat-export.firefox.bookmarklet.oneliner.js`
  - Firefox で使う現在の正式版

## アーカイブしたもの

- `bookmarklets/`
  - 過去の高機能 bookmarklet
- `variants/`
  - `no-obs` や `encoded` などの派生版
- `dist/`
  - Console / Snippets 用や生成途中の minified JS
- `loaders/`
  - GitHub Pages ローダー版

## 方針

- `root` と `docs/` には、実運用で使う bookmarklet だけを残します
- 比較用や調査用の生成物は、この `archive/` に退避します
- 必要になった場合だけ参照し、通常利用では触らない前提です
